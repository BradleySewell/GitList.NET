Gitlist

A Git tool built to easily manage multiple repositories at the same time, change the branch of multiple repositories all at the same time, fetch, reset etc.

A Git tool built to easily manage (change branch, fetch, reset etc.) multiple git repositories at the same time, provided that they are in the same root folder. This is really handy for setting all of your repositories to a specific branch with the click of a button.

  • Also manage individual repositories.

  • Add multiple root directories and it will auto discover all the repositories contained within and group them together.

  • A good overview of each repository, displays what branch they are on and how many commits ahead or behind and how many changes have been made.

  • Each repository has its own git terminal for added flexibility.

  • Auto refresh, periodic refresh, etc.
