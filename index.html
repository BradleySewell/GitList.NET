Gitlist

A Git tool built to easily manage multiple repositories at the same time, change the branch of multiple repositories all at the same time, fetch, reset etc.

View on GitHub Download .zip Download .tar.gz

A Git tool with a good overview and the ability to easily manage (change branch, reset, fetch etc.) multiple git repositories and branches at the same time, provided they are in the same root folder. Add multiple root directories and it will auto discover all the repositories below and group them together. Displays what branch each repository is on and how many commits ahead or behind, also contains a built in git terminal and has other features such as auto refresh etc.
